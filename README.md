# Z-Image-app Monorepo

This repo is a monorepo for the Z-Image-app project.

- Backend: Python FastAPI + Celery
- Frontend: Vite + React + Tailwind CSS v4
- Databases: PostgreSQL 17 + Redis 7

See `docs/architecture.md` for directory layout and `docs/quickstart.md` for a project-wide quick start (API + Worker + Web).
